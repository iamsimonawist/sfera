# SFERA
This repository contains the complete replication package accompanying my Master's thesis "SFERA: A Systematic Framework for the Evaluation of Reference Architectures in Software Systems"

The repository is organised as follows:

## 1_SFERA_RelatedWork
Existing literature relevant to the evaluation of software reference architectures.

## 2_SFERA_InitialSearch
Search string and initial identification of potentially relevant primary studies.

## 3_SFERA_DataExtraction
Data extraction, data analysis and classification, and final list of primary studies.

## 4_SFERA_CaseStudy
Evaluation results for three reference architectures and SFERA itself.
